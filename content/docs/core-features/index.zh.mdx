# SQL Workspace

Dory 仍然是快速、熟悉的人类 SQL 客户端。同一个工作台也会接收 Agent 创建的 SQL 和结果，让你无需重建聊天上下文就能审阅和继续自动化工作。

## 日常 SQL 功能

| 功能 | 用途 |
| --- | --- |
| [Connections](/zh/docs/core-features/connections) | 管理数据库凭据、网络设置和本地文件。 |
| [Explorer](/zh/docs/core-features/explorer) | 检查 Database、Schema、Table、Column、Preview 和对象详情。 |
| [SQL Console](/zh/docs/core-features/sql-console) | 在多标签页中编写、运行、调试和保存 SQL。 |
| [Charts & Results](/zh/docs/core-features/charts-results) | 检查真实行、筛选结果并创建图表。 |
| [AI Chat](/zh/docs/core-features/ai-chat) | 在 Dory 内生成、解释、修复或调整 SQL。 |
| [Saved Queries](/zh/docs/core-features/saved-queries) | 将可复用 SQL 沉淀为长期资产。 |

## ClickHouse 运维

- [Monitoring](/zh/docs/core-features/monitoring)：检查活动、延迟、慢查询和错误。
- [Privileges](/zh/docs/core-features/privileges)：管理 ClickHouse 用户、角色、权限和集群授权。

## AI Chat 与外部 Agent

AI Chat 在 Dory UI 内使用当前连接、标签页、Schema 和结果上下文。外部 Agent 则通过 MCP 或 Dory CLI 接入，创建 Agent Run，并保留 SQL 与结果供人类交接。参阅 [Agent Workspaces](/zh/docs/agents)。
