# 概览

Dory 的主要工作区由 SQL Console、AI Chat、Explorer、结果图表、连接管理和 Saved Queries 组成。

## 功能地图

| 功能 | 用途 |
| --- | --- |
| [SQL Console](/zh/docs/core-features/sql-console) | 编写、运行、调试和保存带 Schema 上下文的 SQL。 |
| [AI Chat](/zh/docs/core-features/ai-chat) | 结合数据库上下文生成 SQL、解释查询、修复错误并继续分析。 |
| [Explorer](/zh/docs/core-features/explorer) | 查看数据库、Schema、表、字段、预览数据和对象详情。 |
| [Charts & Results](/zh/docs/core-features/charts-results) | 检查结果表，并把适合图表的 SQL 输出转成可视化摘要。 |
| [Saved Queries](/zh/docs/core-features/saved-queries) | 保存有价值的 SQL 和可复用分析。 |
| [Connections](/zh/docs/core-features/connections) | 管理数据库连接配置和本地文件路径。 |
| [Monitoring](/zh/docs/core-features/monitoring) | 查看 ClickHouse 查询活动、慢查询、错误查询和运行信号。 |
| [Privileges](/zh/docs/core-features/privileges) | 管理 ClickHouse 用户、角色、权限和集群授权。 |
