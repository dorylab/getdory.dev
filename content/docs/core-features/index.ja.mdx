# SQL Workspace

Dory は人間向けの高速で使い慣れた SQL クライアントです。同じ workspace にエージェントの SQL と結果も届くため、チャットを再構築せずに自動化された作業をレビューして続けられます。

## 日常の SQL 機能

| 機能 | 用途 |
| --- | --- |
| [Connections](/ja/docs/core-features/connections) | DB credential、network、local file の管理。 |
| [Explorer](/ja/docs/core-features/explorer) | Database、Schema、table、column、preview の確認。 |
| [SQL Console](/ja/docs/core-features/sql-console) | 複数タブで SQL を作成、実行、修正、保存。 |
| [Charts & Results](/ja/docs/core-features/charts-results) | 実際の行、filter、chart の確認。 |
| [AI Chat](/ja/docs/core-features/ai-chat) | Dory 内で SQL を生成、説明、修正、変換。 |
| [Saved Queries](/ja/docs/core-features/saved-queries) | 再利用する SQL の保存。 |

## ClickHouse Operations

- [Monitoring](/ja/docs/core-features/monitoring)：activity、latency、slow query、error。
- [Privileges](/ja/docs/core-features/privileges)：user、role、permission、cluster grant。

## AI Chat と外部エージェント

AI Chat は Dory UI 内で現在の接続、タブ、Schema、結果を使います。外部エージェントは MCP または Dory CLI から Agent Run を作成し、人間向けに SQL と結果を保存します。[Agent Workspaces](/ja/docs/agents)を参照してください。
