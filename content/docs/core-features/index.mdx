# Overview

Dory's main workspace includes SQL Console, AI Chat, Explorer, result charts, connection management, and Saved Queries.

## Feature Map

| Feature | Use it for |
| --- | --- |
| [SQL Console](/en/docs/core-features/sql-console) | Write, run, debug, and save SQL with schema context. |
| [AI Chat](/en/docs/core-features/ai-chat) | Generate SQL, explain queries, fix errors, and continue analysis with database context. |
| [Explorer](/en/docs/core-features/explorer) | Inspect databases, schemas, tables, columns, sample rows, and object details. |
| [Charts & Results](/en/docs/core-features/charts-results) | Review result tables and turn chart-ready SQL output into visual summaries. |
| [Saved Queries](/en/docs/core-features/saved-queries) | Save useful SQL and reusable analysis for yourself or a team. |
| [Connections](/en/docs/core-features/connections) | Manage database connection settings and local file paths. |
| [Monitoring](/en/docs/core-features/monitoring) | Inspect ClickHouse query activity, slow queries, errors, and operational signals. |
| [Privileges](/en/docs/core-features/privileges) | Manage ClickHouse users, roles, permissions, and cluster-aware grants. |
