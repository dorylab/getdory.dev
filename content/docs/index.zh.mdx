# Dory 文档

Dory 是面向人类和 Agent 的 AI 原生 SQL 客户端。人类可以查询、探索和可视化数据，AI Agent 则可以通过 MCP 安全访问数据库，并把工作留在同一个可编辑 SQL 工作台中。

## 人类和 Agent 共用一个工作台

人类和 Agent 使用同一套底层数据库动作：

- **人类**编写 SQL、浏览 Schema、检查结果表、创建图表并保存有用查询。
- **Agent** 通过 MCP 列出连接、探索 Schema、运行只读 SQL 并组织分析过程。
- **共享工作台**保留 SQL 标签页、结果快照、图表、保存查询和执行上下文，让人类可以检查、编辑、重新运行并继续工作。

## Dory 与普通数据库 MCP Server

| 能力 | 普通数据库 MCP Server | Dory |
| --- | ---: | ---: |
| 运行 SQL、探索 Schema | 支持 | 支持 |
| 可编辑 SQL 标签页与持久结果 | 不支持 | 支持 |
| 图表、筛选与保存查询 | 有限 | 支持 |
| 人类审阅与继续工作 | 有限 | 支持 |
| 用作日常 SQL 客户端 | 不支持 | 支持 |
