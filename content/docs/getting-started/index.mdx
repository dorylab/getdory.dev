# Overview

The first Dory session has a short path: launch the app, connect a reachable data source, run a limited query, and review the result.

## Setup Path

1. [Quick Start](/en/docs/getting-started/quick-start): choose hosted demo, desktop app, or Docker.
2. [Installation](/en/docs/getting-started/installation): install the desktop app or start a self-hosted runtime.
3. [Connect Your Database](/en/docs/getting-started/connect-your-database): create a connection and confirm schema visibility.
4. [First Query in 3 Minutes](/en/docs/getting-started/first-query-in-3-minutes): run a limited query, review results, and use AI for the next step.

## What Comes After Setup

After the first query works, use the rest of the documentation by intent:

| Need | Read |
| --- | --- |

## Minimum Production Advice

- Use read-only credentials for production databases.
- Confirm network access from the runtime that runs Dory, especially in Docker.
- Review AI-generated SQL before running it.
