# Introduction

Dory is an AI-native data workspace for modern databases. It combines a schema-aware SQL console, conversational AI, database exploration, saved queries, and charting in one workspace.

## What you can do with Dory

- Connect ClickHouse, PostgreSQL, MySQL, SQLite, and other supported databases.
- Explore tables and columns before writing SQL.
- Write, run, and refine SQL with editor assistance.
- Use AI Chat to generate, explain, and debug queries with schema context.
- Save analysis and turn query results into charts.
