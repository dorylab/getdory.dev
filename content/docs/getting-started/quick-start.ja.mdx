# クイックスタート

目的に合う最短の手順を選んでください。

## ライブ SQL ワークスペースを試す

[Dory ライブデモ](https://app.getdory.dev)を開き、**Sign in as demo** を選びます。インストールなしで Schema 探索、SQL 実行、結果確認、チャート作成を試せます。

## ローカル SQL ワークスペースを開始する

```bash
brew install dorylab/dory/dory
```

macOS または Windows 版は [GitHub Releases](https://github.com/dorylab/dory/releases)からもダウンロードできます。その後、Dory を開き、読み取り専用資格情報で接続を作成し、[データベース接続](/ja/docs/getting-started/connect-your-database)へ進みます。

## ローカル エージェントを接続する

Dory Desktop の **Settings → Agent Access** で MCP を有効にし、表示されたエンドポイントをクライアントへ追加します。既定値は次のとおりです。

```text
http://127.0.0.1:3318/api/mcp
```

```bash
codex mcp add dory --url http://127.0.0.1:3318/api/mcp
codex mcp list
```

Desktop の認可は Dory が管理するため、通常のユーザーは Bearer Token をコピーする必要がありません。standalone、headless、hosted の構成は [MCP セットアップ](/ja/docs/agents/mcp-setup)を参照してください。

## Dory Web を実行する

チームがブラウザーから利用する場合は Web Docker イメージを使います。必須シークレット、永続化、公開 URL を扱う [Docker ガイド](/ja/docs/deploy/docker)から開始してください。

## 完了条件

SQL ワークスペースまたは接続済み MCP クライアントが利用できれば完了です。[3 分で最初のクエリ](/ja/docs/getting-started/first-query-in-3-minutes)または [Agent Runs](/ja/docs/agents/agent-runs)へ進みます。
