# 安装

Dory 为交互式 SQL 工作、共享 Web 部署和纯 Agent Runtime 提供不同入口。

## Dory Desktop

在 macOS 上通过 Homebrew 安装：

```bash
brew install dorylab/dory/dory
```

也可以从 [GitHub Releases](https://github.com/dorylab/dory/releases) 下载 macOS 和 Windows 安装包。Desktop 适合日常 SQL 工作、本地数据库文件和 Desktop MCP。

## Dory Web

需要浏览器界面和团队共享工作台时使用 `dorylab/dory` 镜像。实际部署必须配置应用密钥、持久化存储和正确的公网 URL，请按照 [Docker](/zh/docs/deploy/docker)文档操作，不要只运行裸容器命令。

## Dory CLI

`@getdory/cli` 可以在不打开桌面应用的情况下运行 Dory，需要 Node.js 20 或更高版本。

```bash
npx -y @getdory/cli --help
```

长期运行的服务器可以全局安装：

```bash
npm install -g @getdory/cli
dory --help
```

CLI 支持 `standalone`、`desktop` 和 `self-hosted` 数据模式。继续阅读 [CLI & Headless Runtime](/zh/docs/agents/cli-headless)。

## Headless Docker

Agent 或自动化需要 HTTP MCP 但不需要 Web UI 时，使用 `dorylab/dory-headless`。Token、持久卷与 TLS 配置请参阅 [CLI & Headless Runtime](/zh/docs/agents/cli-headless)。

## 验证安装

- Desktop 可以打开并进入 [Connections](/zh/docs/core-features/connections)。
- Web 拥有持久化应用存储和正确的登录 URL。
- CLI 初始化后可以通过 `dory doctor --data standalone` 检查。
- Headless HTTP 使用 Bearer Token 保护，且未在没有 TLS 的情况下暴露到公网。
