# Inicio rápido

Elige la ruta más corta para el resultado que buscas.

## Prueba el espacio SQL en línea

Abre la [demo de Dory](https://app.getdory.dev), selecciona **Sign in as demo** y usa los datos incluidos para explorar el esquema, ejecutar SQL, revisar resultados y crear un gráfico. No requiere instalación.

## Inicia un espacio SQL local

```bash
brew install dorylab/dory/dory
```

También puedes descargar versiones para macOS o Windows desde [GitHub Releases](https://github.com/dorylab/dory/releases). Después, abre Dory, crea una conexión con credenciales de solo lectura y continúa con [Conecta tu base de datos](/es/docs/getting-started/connect-your-database).

## Conecta un agente local

En Dory Desktop abre **Settings → Agent Access**, activa MCP y añade el endpoint mostrado a tu cliente. El valor predeterminado es:

```text
http://127.0.0.1:3318/api/mcp
```

```bash
codex mcp add dory --url http://127.0.0.1:3318/api/mcp
codex mcp list
```

Dory administra la autorización de Desktop, por lo que un usuario normal no necesita copiar un Bearer Token. Para standalone, headless o hosted, consulta [MCP Setup](/es/docs/agents/mcp-setup).

## Ejecuta Dory Web

Usa la imagen Web de Docker cuando un equipo necesite acceso desde el navegador. Empieza con la [guía de Docker](/es/docs/deploy/docker), que cubre secretos, persistencia y URL pública.

## Resultado esperado

Debes terminar con un espacio SQL funcional o un cliente MCP conectado. Continúa con [Primera consulta en 3 minutos](/es/docs/getting-started/first-query-in-3-minutes) o [Agent Runs](/es/docs/agents/agent-runs).
