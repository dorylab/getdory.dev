# インストール

Dory には、対話型 SQL、共有 Web、エージェント専用 Runtime それぞれのエントリーポイントがあります。

## Dory Desktop

```bash
brew install dorylab/dory/dory
```

macOS と Windows のインストーラーは [GitHub Releases](https://github.com/dorylab/dory/releases)から入手できます。Desktop は日常の SQL、ローカル データベース ファイル、Desktop MCP に適しています。

## Dory Web

ブラウザー UI と共有チーム ワークスペースには `dorylab/dory` を使用します。実運用ではシークレット、永続ストレージ、正しい公開 URL が必要です。[Docker](/ja/docs/deploy/docker)の手順に従ってください。

## Dory CLI

`@getdory/cli` は Desktop を開かずに Dory を実行します。Node.js 20 以降が必要です。

```bash
npx -y @getdory/cli --help
```

長時間動かすサーバーではグローバル インストールできます。

```bash
npm install -g @getdory/cli
dory --help
```

CLI は `standalone`、`desktop`、`self-hosted` のデータ モードをサポートします。[CLI & Headless Runtime](/ja/docs/agents/cli-headless)へ進んでください。

## Headless Docker

Web UI なしでエージェントや自動化に HTTP MCP を提供する場合は `dorylab/dory-headless` を使用します。Token、永続ボリューム、TLS は [CLI & Headless Runtime](/ja/docs/agents/cli-headless)を参照してください。

## 確認

- Desktop から [Connections](/ja/docs/core-features/connections)を開ける。
- Web に永続ストレージと正しいサインイン URL がある。
- CLI の初期化後に `dory doctor --data standalone` が成功する。
- Headless HTTP は Bearer Token で保護され、TLS なしで公開されていない。
