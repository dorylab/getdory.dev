# 概览

Dory 的首次使用路径很短：先启动应用，再连接一个可访问的数据源，最后运行一条有限制的查询并检查结果。

## 入门路径

1. [快速开始](/zh/docs/getting-started/quick-start)：选择在线演示、桌面端或 Docker。
2. [安装说明](/zh/docs/getting-started/installation)：安装桌面端或启动自托管运行环境。
3. [连接数据库](/zh/docs/getting-started/connect-your-database)：创建连接并确认 Schema 可见。
4. [3 分钟完成第一条查询](/zh/docs/getting-started/first-query-in-3-minutes)：运行有限制的查询、检查结果，并让 AI 继续辅助下一步。

## 生产数据前的最低建议

- 生产数据库优先使用只读账号。
- Docker 中连接本地文件或内网数据库时，先确认运行环境真的能访问。
- AI 生成的 SQL 执行前需要人工审查。
