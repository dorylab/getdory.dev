# 快速开始

根据你的目标选择最短路径。

## 体验在线 SQL 工作台

打开 [Dory 在线 Demo](https://app.getdory.dev)，选择 **Sign in as demo**，使用内置数据探索 Schema、运行 SQL、检查结果并创建图表，无需安装。

## 启动本地 SQL 工作台

安装 Dory Desktop：

```bash
brew install dorylab/dory/dory
```

也可以从 [GitHub Releases](https://github.com/dorylab/dory/releases) 下载 macOS 或 Windows 版本。然后：

1. 打开 Dory。
2. 使用只读凭据创建连接。
3. 继续阅读[连接数据库](/zh/docs/getting-started/connect-your-database)。

## 连接本地 Agent

如果已经安装 Dory Desktop，打开 **Settings → Agent Access**，启用 MCP，并将显示的端点添加到客户端。默认端点是：

```text
http://127.0.0.1:3318/api/mcp
```

Codex CLI 示例：

```bash
codex mcp add dory --url http://127.0.0.1:3318/api/mcp
codex mcp list
```

Dory 会管理 Desktop 授权，普通 Desktop 用户无需复制 Bearer Token。standalone、headless 或 hosted 接入方式请阅读 [MCP Setup](/zh/docs/agents/mcp-setup)。

## 运行 Dory Web

团队需要浏览器访问时使用 Web Docker 镜像。请从[Docker 指南](/zh/docs/deploy/docker)开始，其中包含必需密钥、持久化和公网 URL 配置。

## 完成标准

你应当得到可用的 SQL 工作台或已连接的 MCP 客户端。接下来阅读[三分钟完成第一个查询](/zh/docs/getting-started/first-query-in-3-minutes)或 [Agent Runs](/zh/docs/agents/agent-runs)。
