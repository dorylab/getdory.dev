# Quick Start

Use this path when you want the fastest way to try Dory and run a first query.

## Option 1: Try the Demo

Open `https://app.getdory.dev` and choose **Enter as Demo**.

This is best when:

- You want to inspect the interface quickly.
- You do not have database credentials ready.
- You want to try SQL Console and AI features first.

## Option 2: Install Locally

Install the desktop app if you want to connect your own databases.

### macOS（Homebrew）

```bash
brew install dorylab/dory/dory
```

After installation, open Dory and connect a database.

### Other Install Options

For Docker or other runtime options, see [Installation](./installation).

## After Install

Follow this sequence:

1. Complete [Connect Your Database](./connect-your-database).
2. Run [First Query in 3 Minutes](./first-query-in-3-minutes).
3. Open [SQL Console](../core-features/sql-console).
4. Use [AI Chat](../core-features/ai-chat) to generate, explain, or optimize SQL.

## Quick Tips

- Start with a test database if this is your first session.
- AI works best after Dory can read schema context.
- If you use DataGrip, DBeaver, or TablePlus, you can start by treating Dory as a modern SQL workspace.
