# Dory Usage Guide

Dory is a SQL and AI data workspace. Use this guide as the high-level path for a first successful session.

## Recommended Flow

1. Open the live demo, desktop app, or self-hosted deployment.
2. Create or choose a workspace.
3. Add a database connection and test it.
4. Inspect databases, schemas, tables, and columns in Explorer.
5. Run a small `LIMIT` query in SQL Console.
6. Use AI Chat or Copilot to generate, explain, fix, or optimize SQL.
7. Validate results, then build a chart if the result shape is suitable.
8. Save useful SQL in Saved Queries for reuse.

## Supported Connection Types

Dory supports ClickHouse, PostgreSQL, Neon, MySQL, MariaDB, SQLite, and DuckDB / MotherDuck.
