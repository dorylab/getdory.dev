# Dory ドキュメント

Dory は人間とエージェントのための AI ネイティブ SQL クライアントです。人間はデータを照会、探索、可視化でき、AI エージェントは MCP を通じて安全にデータベースを操作し、その作業を同じ編集可能な SQL ワークスペースに残せます。

## 目的から選ぶ

| 目的 | 開始ページ |
| --- | --- |
| 日常の SQL クライアントとして使う | [クイックスタート](/ja/docs/getting-started/quick-start)から、[データベース接続](/ja/docs/getting-started/connect-your-database)、[最初のクエリ](/ja/docs/getting-started/first-query-in-3-minutes)へ進みます。 |
| Codex、Claude Code、その他のエージェントにデータベースを使わせる | [Agent Workspaces](/ja/docs/agents)で概要を確認し、[MCP セットアップ](/ja/docs/agents/mcp-setup)を選びます。 |
| エージェントが作成した作業をレビューする | [Agent Run と人間への引き継ぎ](/ja/docs/agents/agent-runs)を参照します。 |
| チーム向けに Dory をデプロイする | [セルフホスティング](/ja/docs/deploy/self-hosting)から開始します。 |

## 人間とエージェントの共有ワークスペース

- **人間**は SQL の作成、Schema の閲覧、結果の確認、チャート作成、クエリ保存を行います。
- **エージェント**は MCP から接続一覧、Schema 探索、読み取り専用 SQL、分析の整理を行います。
- **共有ワークスペース**には SQL タブ、結果スナップショット、チャート、保存済みクエリ、実行コンテキストが残り、人間が確認、編集、再実行、継続できます。

![Dory はエージェントのデータベース作業を編集可能な SQL ワークスペースに変換します](/actions.png)

## Dory と一般的なデータベース MCP Server

| 機能 | 一般的な DB MCP Server | Dory |
| --- | ---: | ---: |
| SQL 実行と Schema 探索 | あり | あり |
| 編集可能な SQL タブと永続的な結果 | なし | あり |
| チャート、フィルター、保存済みクエリ | 限定的 | あり |
| 人間によるレビューと継続 | 限定的 | あり |
| 日常の SQL クライアント | なし | あり |
