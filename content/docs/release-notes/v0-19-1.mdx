# v0.19.1

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.19.1.md)

## What's Changed

### Features
- Added desktop MCP (Model Control Protocol) grants with automatic token UI hiding for improved security and streamlined agent access.
- Introduced per-user MCP proxy restoration when switching desktop accounts, ensuring consistent agent permissions across sessions.

### Improvements
- Refined MCP settings UI: updated labels, layout, and organization in the Agent Access panel for clarity and usability.
- Updated billing and settings pages with improved localization support across English, Spanish, Japanese, and Chinese.

### Fixes
- Removed unusable offline authentication flows, eliminating misleading UI elements (e.g., offline mode banner) and deprecated API routes—improving reliability and reducing confusion during sign-in.

### Internal
- Consolidated i18n assets across web and shared packages for consistent localization handling.
- Cleaned up stale release notes (`v0.18.0.md`) and merged related feature branches (`#252`, `#253`) to streamline release history.
