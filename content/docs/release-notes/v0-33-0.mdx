# v0.33.0

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.33.0.md)

## What's Changed

### Improvements
- Refined application workflows and user experience across database, schema, and table exploration interfaces—particularly in schema graph visualization, database/schema view rendering, and agent-run summaries.
- Streamlined table explorer panels and inspector portal handling, improving consistency and responsiveness when interacting with table columns, indexes, and data previews.

### Fixes
- Resolved inconsistencies in fallback view rendering for databases and schemas, ensuring reliable display when primary views fail to load.

### Internal
- Removed outdated release notes (`v0.31.0.md`) and updated `v0.32.0.md` to reflect current documentation standards.
