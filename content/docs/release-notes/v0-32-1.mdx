# v0.32.1

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.32.1.md)

## What's Changed

### Fixes
- Fixed SQL console hydration issues and improved editor fallback rendering, resolving cases where the SQL editor failed to load or displayed incorrectly after page navigation or refresh.

### Improvements
- Clarified local AI setup instructions across desktop and web settings pages, with updated UI text and localized guidance in English, Spanish, Japanese, and Chinese.

### Internal
- Removed legacy `pglite` assets from the Docker build process, reducing image size and simplifying the container configuration.
