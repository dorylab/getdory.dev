# v0.32.0

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.32.0.md)

## What's Changed

### Improvements
- Docker images now include artifacts and resultset package manifests, ensuring consistent and reproducible builds with all required runtime dependencies.

### Fixes
- Added regression tests for result set migration synchronization, improving reliability and correctness when upgrading database schemas related to result sets.

### Internal
- Introduced two new database migrations (`0019_resultset_artifacts` and `0020_resultset_schema_updates`) across both PostgreSQL and pgLite backends to support result set artifact storage and schema evolution.
