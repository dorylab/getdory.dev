# v0.33.1

[View source on GitHub](https://github.com/dorylab/dory/blob/main/release-notes/v0.33.1.md)

## What's Changed

### Features
- Added a new **Dory CLI settings panel** in the web app’s organization settings, enabling users to configure and manage Dory CLI integration directly from the UI.
- Added **multi-language support** for the Dory CLI settings panel, with translations available in English, Spanish, Japanese, and Chinese.

### Improvements
- Enhanced **SQL autocompletion** in the SQL console to better handle partial JOIN clauses and table/column aliases—improving accuracy and developer productivity when writing complex queries.

### Documentation
- Added documentation and a visual diagram for the **Schema Explorer**, helping users understand how database schemas are represented and navigated in the application.

### Internal
- Refactored application workflows across packages (`web`, `cli`, `mcp`, `server-core`) to simplify implementation and improve maintainability.
