# データベース

Dory は複数データベース対応の SQL クライアント兼 Agent workspace です。すべての接続で Schema 探索、SQL、結果、作業保存を利用でき、ClickHouse には深い運用機能があります。

| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/postgresql.svg" alt="" width="20" height="20" />[PostgreSQL](/ja/docs/database-support/postgresql)</span> | 対応 | SQL、Schema、結果、AI、Agent。 |

[Connections](/ja/docs/core-features/connections)で作成し、[Explorer](/ja/docs/core-features/explorer)で実際の Schema を確認します。人間は [SQL Console](/ja/docs/core-features/sql-console)、エージェントは [MCP Setup](/ja/docs/agents/mcp-setup)から同じ接続を利用できます。

TLS、IP allowlist、SSH Tunnel は [Cloud Databases](/ja/docs/database-support/cloud-databases)を参照してください。
