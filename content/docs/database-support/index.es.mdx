# Bases de datos

Dory es un cliente SQL multibase y espacio para agentes. Todas las conexiones admiten explorar esquema, ejecutar SQL, revisar resultados y conservar trabajo; ClickHouse añade operaciones más profundas.

## Matriz de compatibilidad

| Base de datos | Estado | Capacidades |
| --- | --- | --- |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/clickhouse.svg" alt="" width="20" height="20" />[ClickHouse](/es/docs/database-support/clickhouse)</span> | Integración profunda | SQL, Explorer, AI, Agent, Monitoring y Privileges. |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/postgresql.svg" alt="" width="20" height="20" />[PostgreSQL](/es/docs/database-support/postgresql)</span> | Compatible | SQL, Schema, resultados, AI y Agent. |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/neon.svg" alt="" width="20" height="20" />[Neon](/es/docs/database-support/neon)</span> | Compatible | PostgreSQL serverless. |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/supabase.svg" alt="" width="20" height="20" />[Supabase](/es/docs/database-support/supabase)</span> | Compatible | Conexión nativa sobre Supabase PostgreSQL. |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/mysql.svg" alt="" width="20" height="20" />[MySQL](/es/docs/database-support/mysql)</span> | Compatible | SQL, Explorer, resultados, AI y Agent. |
| <span class="dory-db-name"><img class="dory-db-icon" src="/icons/databases/mariadb.svg" alt="" width="20" height="20" />[MariaDB](/es/docs/database-support/mariadb)</span> | Compatible | Conexión y dialecto MariaDB. |

Crea una conexión en [Connections](/es/docs/core-features/connections), inspecciona el esquema en [Explorer](/es/docs/core-features/explorer) y usa [SQL Console](/es/docs/core-features/sql-console) o [MCP Setup](/es/docs/agents/mcp-setup). Consulta [Cloud Databases](/es/docs/database-support/cloud-databases) para TLS, allowlists y SSH Tunnel.
