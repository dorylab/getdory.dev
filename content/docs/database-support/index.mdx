# Databases

Dory is a multi-database SQL client and agent workspace. Every supported connection participates in the same core flow: connect, explore schema, run SQL, inspect results, and preserve work. ClickHouse also includes deeper operations surfaces.

## Human and Agent Flow

1. Create and test a connection in [Connections](/en/docs/core-features/connections).
2. Inspect real objects in [Explorer](/en/docs/core-features/explorer).
3. Write SQL in [SQL Console](/en/docs/core-features/sql-console) or connect an agent through [MCP Setup](/en/docs/agents/mcp-setup).
4. Review SQL and results in the shared workspace.
5. Save reusable SQL or keep the evidence in an Agent Run.

For managed services, private networks, TLS, allowlists, and SSH tunnels, see [Cloud Databases](/en/docs/database-support/cloud-databases).
