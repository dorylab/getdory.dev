# 数据库

Dory 是多数据库 SQL 客户端和 Agent 工作台。所有支持的连接都可以完成连接、探索 Schema、运行 SQL、检查结果和保留工作；ClickHouse 还提供更深入的运维界面。

## 人类与 Agent 工作流

1. 在 [Connections](/zh/docs/core-features/connections) 中创建并测试连接。
2. 在 [Explorer](/zh/docs/core-features/explorer) 中检查真实对象。
3. 在 [SQL Console](/zh/docs/core-features/sql-console) 编写 SQL，或通过 [MCP Setup](/zh/docs/agents/mcp-setup)连接 Agent。
4. 在共享工作台审阅 SQL 和结果。
5. 将可复用 SQL 保存为 Saved Query，或把证据留在 Agent Run 中。

托管服务、私有网络、TLS、IP allowlist 和 SSH Tunnel 请参阅[Cloud Databases](/zh/docs/database-support/cloud-databases)。
