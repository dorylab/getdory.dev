# Workflows

Feature 页面解释单个产品功能，这些指南则把功能连接成面向结果的可重复流程。

## SQL Workspace 工作流

| 工作流 | 结果 |
| --- | --- |
| [探索陌生数据表](/zh/docs/workflows/explore-unknown-tables) | 使用 Explorer、预览和 AI Chat 理解陌生 Schema。 |
| [使用 Ask AI 生成 SQL](/zh/docs/workflows/generate-sql-with-ai) | 在当前工作台把问题转成可审阅 SQL。 |
| [调试慢查询](/zh/docs/workflows/debug-slow-queries) | 使用 SQL Console、AI 和 ClickHouse Monitoring 调查高成本 SQL。 |
| [从 SQL 创建图表](/zh/docs/workflows/build-charts-from-sql) | 把查询输出整理为清晰的可视化结果。 |
| [保存并复用分析](/zh/docs/workflows/save-reuse-analysis) | 将稳定 SQL 沉淀为带说明的可复用资产。 |

## Agent 工作流

| 工作流 | 结果 |
| --- | --- |
| [使用 Agent 分析数据库](/zh/docs/workflows/agent-database-analysis) | 探索真实 Schema、运行只读 SQL 并保留可审计 Agent Run。 |
| [审阅并继续 Agent 工作](/zh/docs/workflows/review-agent-work) | 验证 Agent 证据、编辑并重跑 SQL，然后继续工作。 |

新用户应先完成[三分钟完成第一个查询](/zh/docs/getting-started/first-query-in-3-minutes)。
