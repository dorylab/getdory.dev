# Workflows

Feature pages explain individual surfaces. These guides connect them into repeatable outcomes for people and agents.

## SQL Workspace Workflows

| Workflow | Outcome |
| --- | --- |
| [Explore Unknown Tables](/en/docs/workflows/explore-unknown-tables) | Understand unfamiliar schema with Explorer, previews, and AI Chat. |
| [Generate SQL with Ask AI](/en/docs/workflows/generate-sql-with-ai) | Turn a question into reviewable SQL inside the current workspace. |
| [Debug Slow Queries](/en/docs/workflows/debug-slow-queries) | Investigate expensive SQL with the Console, AI, and ClickHouse Monitoring. |
| [Build Charts from SQL](/en/docs/workflows/build-charts-from-sql) | Shape query output for clear visual review. |
| [Save & Reuse Analysis](/en/docs/workflows/save-reuse-analysis) | Turn stable SQL into a documented reusable asset. |

## Agent Workflows

| Workflow | Outcome |
| --- | --- |
| [Analyze a Database with an Agent](/en/docs/workflows/agent-database-analysis) | Explore real schema, run bounded read-only SQL, and preserve an auditable Agent Run. |
| [Review and Continue Agent Work](/en/docs/workflows/review-agent-work) | Verify agent evidence, edit SQL, rerun it, and continue in the normal workspace. |

New users should complete [First Query in 3 Minutes](/en/docs/getting-started/first-query-in-3-minutes) before choosing a longer workflow.
