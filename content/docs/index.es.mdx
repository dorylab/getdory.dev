# Documentación de Dory

Dory es un cliente SQL nativo de IA para personas y agentes. Las personas pueden consultar, explorar y visualizar datos, mientras que los agentes de IA trabajan de forma segura con bases de datos mediante MCP y conservan su trabajo en el mismo espacio SQL editable.

## Un espacio para personas y agentes

- **Las personas** escriben SQL, exploran esquemas, revisan resultados, crean gráficos y guardan consultas útiles.
- **Los agentes** enumeran conexiones, exploran esquemas, ejecutan SQL de solo lectura y organizan análisis mediante MCP.
- **El espacio compartido** conserva pestañas SQL, resultados, gráficos, consultas guardadas y contexto de ejecución para que una persona pueda revisar, editar, volver a ejecutar y continuar el trabajo.

## Dory frente a un servidor MCP de base de datos

| Capacidad | Servidor MCP convencional | Dory |
| --- | ---: | ---: |
| Ejecutar SQL y explorar esquemas | Sí | Sí |
| Pestañas SQL editables y resultados persistentes | No | Sí |
| Gráficos, filtros y consultas guardadas | Limitado | Sí |
| Revisión y continuación por una persona | Limitado | Sí |
| Cliente SQL para uso diario | No | Sí |
